OutlookManager 使用协议

1. 本系统仅可在合法、授权和合规场景下使用，不得用于未获授权的邮箱访问、数据收集、数据滥用或其他违法违规操作。

2. 你应自行保管管理员密码、Refresh Token、Client ID、API Key 等敏感信息，并对全部调用、管理与导出行为负责。

3. 若你将本系统暴露在公网环境中，应自行完成 HTTPS、网络访问控制、审计日志、备份和最小权限等安全措施。

4. API Key 功能用于程序化调用接口。部署者应只向可信系统发放密钥，并及时回收失效、泄露或超期的密钥。

5. 任何因部署不当、密钥泄露、权限管理缺失、第三方滥用或违规接入造成的数据泄露、账号封禁、业务损失或法律风险，均由部署者自行承担。

6. 继续使用即表示你已阅读、理解并接受以上条款，并确认你有权管理当前部署环境及相关 Outlook 数据。

API 密钥

创建并管理用于接口调用的 API Key，支持永不过期、固定到期时间、无限请求和固定调用次数。

创建新 API Key

创建后会返回一次完整密钥，请立即保存。后续页面只显示前缀和使用统计，不会再次显示完整值。

认证方式

接口支持以下任一请求头携带 API Key。

Authorization: Bearer om_your_api_key
X-API-Key: om_your_api_key

每次通过 API Key 的接口请求都会记录一次调用，并从剩余额度中扣除 1 次。

GET /accounts

获取系统中已添加的全部邮箱账户列表，支持分页与模糊搜索。

请求参数

page integer 页码，从 1 开始，默认值为 1
page_size integer 每页数量，范围 1-100，默认值为 10
email_search string 邮箱账号模糊搜索，可选
tag_search string 标签模糊搜索，可选

响应示例

{ "total_accounts": 25, "page": 1, "page_size": 10, "total_pages": 3, "accounts": [ { "email_id": "example@outlook.com", "client_id": "your-client-id", "status": "active", "tags": ["宸ヤ綔", "涓汉"] } ] }

获取指定邮箱的邮件列表，支持分页和筛选。


                                    

                                        
路径参数

                                        

                                            

                                                email_id
                                                string
                                                邮箱地址，需要进行 URL 编码
                                            

                                        

                                    


                                    

                                        
查询参数

                                        

                                            

                                                folder
                                                string
                                                邮件文件夹（all、inbox、junk），默认值为 all

                                                integer
                                                页码，从 1 开始，默认值为 1
                                            

                                            

                                                page_size
                                                integer
